## 快速重點

濕潤氣流迎山上升 → 冷卻凝結降雨 → 背風坡下降絕熱增溫變乾 → 形成焚風。

主要公式：氣溫變化率

• 乾絕熱遞減率（上升或下降無凝結）：$\dfrac{dT}{dz} = \pm1\ \mathrm{^\circ C/100m}$

• 濕絕熱遞減率（凝結放熱階段）：$\dfrac{dT}{dz} = -0.6\ \mathrm{^\circ C/100m}$


## 解釋/定義

焚風是一種越過山脈後在背風坡形成的乾熱風，因氣流絕熱壓縮升溫所造成。

• 又稱「落山風」或「Foehn wind」。

• 為局部性熱風現象。


## 詳細說明

成因過程 🌬️：

• 濕潤氣團遇山被迫上升 → 溫度下降。

• 達露點後開始凝結、降水 → 放出潛熱。

• 水氣流失後跨過山頂下降 → 氣壓增大 → 絕熱增溫。

• 因乾絕熱遞增率（$1\ \mathrm{^\circ C/100m}$）大於濕絕熱遞減率（$0.6\ \mathrm{^\circ C/100m}$），背風坡溫度高於迎風坡。

特徵：

• 背風坡氣候乾燥、高溫、晴朗。

• 常引起農作物乾枯、森林火災等現象。

全球實例 🌏：

• 阿爾卑斯山焚風（Foehn）— 名稱起源地。

• 北美落磯山「奇努克風」（Chinook）。

• 日本「颪風」、台灣中部山區「落山風」皆屬焚風型態。


## 常見考點/易錯點

命題焦點：

• 上升與下降氣流溫度變化比較。

• 凝結放熱與絕熱增溫的能量平衡。

• 為何背風坡溫度較迎風坡高。

## 常見考點/易錯點

• 誤以為焚風是外來高溫氣團吹入造成 ❌。

• 忽略凝結放熱與氣團失水的作用。

關聯公式：

• 絕熱方程式：$P V^{\gamma} = \text{常數}$，其中 $\gamma = \dfrac{C_p}{C_v}$。

• 溫度垂直梯度：$\Gamma_d = 9.8\ \mathrm{K/km}$。


## 舉例說明

模擬考題

題目：一氣團自山腳（氣溫 $25\ \mathrm{^\circ C}$）上升至山頂（2000 m），達露點 $15\ \mathrm{^\circ C}$ 並降雨，試求背風坡下降至山腳的氣溫。

解答：

• 上升前 $1000\ \mathrm{m}$ 為乾絕熱冷卻：$25 - 10 = 15\ \mathrm{^\circ C}$。

• 再上升 $1000\ \mathrm{m}$ 為濕絕熱冷卻：$15 - 6 = 9\ \mathrm{^\circ C}$。

• 下降 $2000\ \mathrm{m}$ 為乾絕熱增溫：$9 + 20 = 29\ \mathrm{^\circ C}$。

• 答：背風坡山腳氣溫為 $29\ \mathrm{^\circ C}$ ✅
